**Crossplatform Brainfuck interpretator on C**

Author: OriginTravel

License: GNU GPL V3.0

**Compile:**
  * Install C/C++ Compiler (gcc, clang, msvc)
  * Install cmake (or manualy compile main.c)
  * run compile.sh (if use cmake)
  * Done!
